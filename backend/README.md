# Workshelf — требования (черновик)

Система предназначена для управления задачами внутри одной организации.

## Роли
- Руководитель (manager)
- Сотрудник (employee)

## Сценарии
1. Руководитель создаёт задачу: название, описание, дедлайн, исполнитель (пользователь или группа).
2. Сотрудник получает уведомление о новой задаче.
3. Сотрудник меняет статус задачи на "Выполнено" и при необходимости оставляет комментарий.
4. Руководитель видит список задач и статусы выполнения.
