# Workshelf

Workshelf — внутренний органайзер задач для сотрудников в рамках одной организации.

## Роли
- Руководитель: создаёт задачи (на сотрудника или группу), отслеживает статус выполнения.
- Сотрудник: видит назначенные задачи, получает уведомления, отмечает выполнение и оставляет комментарий (опционально).

## Основной функционал (план)
- Авторизация (логин/пароль или авто-логин через токены — уточняется)
- Задачи: создание, назначение, дедлайны, статусы
- Комментарии к задачам
- Уведомления о новых задачах/изменениях
- Панель руководителя для контроля выполнения

## Структура репозитория
- `frontend/` — клиент (React)
- `backend/` — сервер (C# / ASP.NET Core Web API)
- `db/` — БД, миграции/скрипты
- `docs/` — требования, API, схема БД
